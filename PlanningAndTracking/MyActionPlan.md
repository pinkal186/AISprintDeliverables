## 🎯 PRIORITY 4: System Design Thinking Templates (Day 2 Morning - 3 hours)

Build reusable templates so you can create audits FAST.

### Task 1: Create Workflow Mapping Template (1 hour)
- [ ] Study Trigger → Decision → Output framework
- [ ] Create template with sections:
  ```
  COMPANY: [name]
  PERSON: [name, role]
  WORKFLOW: [which process]
  
  TRIGGER: What starts this process?
  DECISION POINTS: What choices do humans make?
  CURRENT OUTPUT: What's produced today?
  BOTTLENECKS: Where does it break/slow down?
  
  AI OPPORTUNITY: What could be automated/augmented?
  HUMAN IN LOOP: Where must humans stay?
  EXPECTED VALUE: Time saved / errors reduced / cost saved
  ```

### Task 2: Create Your First 3 Template Audits (2 hours)
- [ ] Pick 3 companies from your domain research
- [ ] Fill template for each (aim for 30-40 min per company)
- [ ] Don't make them perfect - make them SPECIFIC
- [ ] Focus on ONE clear workflow per company

### Output Files:
- `System_Audit_Template.md`
- `Audits/[CompanyName]_Audit.md` (3 files)

**Reference**: See Week 3 Deliverable 4 in [summary.md](AI_Sprint_Deliverables_Summary.md#deliverable-4-system-design--cold-audit-outreach-15-people)
